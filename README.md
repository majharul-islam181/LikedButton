# Likebutton - Animation
(A new Flutter project.)


## Add this dependency 
```
 like_button: ^2.0.4
```

# Dart file

```dart
LikeButton(
          size: 80,
          animationDuration: Duration(milliseconds: 4000),
          likeCount: 120,

          countPostion: CountPostion.bottom,
        ),
```
